# 变更记录

本项目从 `0.2.0` 起采用此文件记录可见变化。

## 0.2.0 - 2026-07-16

### 新增

- 标准 `src/` 包结构、应用工厂、CLI、WSGI 入口和自动化测试。
- 安全的 Annoy JSON 文档映射、旧索引迁移脚本和资产校验清单。
- 按 `session_id` 隔离的短期记忆。
- 插件清单、OpenAPI、示例文件和动态部署主机。
- 索引构建、文档总结、CAJ 转换和 PDF 合并工具。
- CI、依赖漏洞审计、Dependabot、贡献与安全文档。

### 兼容

- 保留 `POST /set_Query`、`POST /set_text`、旧响应字段和入口欢迎文本。
- 保留两级提示路由、13 个物理子领域及历史提示词文本。

### 修复

- 移除源码明文凭据和导入时模型、pickle、索引副作用。
- 修复未定义上传文件、固定临时文件、错误文档读取及跨用户共享记忆。
- 为远程文档下载增加 SSRF、超时、重定向和大小限制。
- 修复插件主机、OpenAPI 响应和重复 YAML 键。

### 安全迁移

- 运行时不再反序列化 LangChain pickle。
- 从已停止维护的旧 LangChain 文心适配链迁移到千帆 v2 接口。
