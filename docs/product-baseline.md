# 产品行为基线

本页把 2024 年产品资料中明确描述的行为，与当前代码能够证明的事实分开记录。

## 正式产品边界

- 目标用户：PT、CYPT、CUPT、IYPT 等物理竞赛参与者，包括高中生、大学生和教师。
- 核心能力：专业问答、文档总结。
- 当前实现属于 RAG、提示路由和短期记忆，不是模型微调平台，也没有持续“自学习”用户反馈。
- 医疗、法律等场景只出现在历史愿景中，当前知识库和提示词不支持把它们作为正式产品范围。

## 必须保留的行为

1. `POST /set_Query` 接收问题并返回问答结果。
2. `POST /set_text` 接收远程文档 URL 并返回总结。
3. 问题经过语种识别、关键词提取、翻译、原文/译文检索和低分关键词补位。
4. 一级路由为物理、总结、生活；物理再进入 13 个专业子路由。
5. 回答最多组合 3 个当前知识库片段和 3 个同会话历史片段。
6. 总结提示要求全篇总结、逐段总结和关键点。
7. 不知道答案时应明确承认，物理回答应尽可能详细、公式化且通俗。

## 历史参数

| 参数 | 权威应用实际值或当前选择 |
|---|---|
| 嵌入模型 | `paraphrase-multilingual-mpnet-base-v2` |
| 维度 | 768 |
| Annoy metric | `dot` |
| 当前知识库阈值 | `0.6`（实际源码） |
| 历史记忆阈值 | `0.58` |
| 模型温度 | `0.9` |
| 默认最终分块 | 1000 字符、80 字符重叠 |
| 默认 Annoy 构建 | 800 trees、8 jobs |

详细方案曾给出另一组实验参数 `3000/300/600/50、1000 trees、6 jobs`。新脚本把参数全部开放为命令行选项，但默认使用权威应用源码的实际值。

## 不能当作当前事实的历史声明

- “已上线”与“尚未上线”的表述在两份资料中相互矛盾。
- 2017–2023 年 119 道赛题、100 余篇论文和 5 本专著是历史资料规模声明，不代表公共仓库随附这些数据。
- 旧手册中的文心一言插件选择流程属于历史演示。
- 原始语料、教材和论文存在版权边界，未确认前不能公开分发。
- 百度翻译回译扩充和持续反馈学习没有形成可验证的当前实现。

原始资料保存在 `docs/reference/`，演示视频保存在 `docs/media/`。
